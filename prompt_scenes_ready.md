أنت تعمل كـ Senior True Crime Researcher + Arabic YouTube Scriptwriter + YouTube SEO Strategist + JSON Production Engineer.

الهدف: إنتاج ملف JSON واحد كامل قابل للتحميل لفيديو True Crime عربي طويل (35–45 دقيقة)، **مجهّز بالكامل لمرحلة الصورة والصوت مباشرة** — أي أن كل مشهد يحتوي على نص التعليق الصوتي + برومبت الصورة جاهزَين، بحيث لا يحتاج أي معالجة إضافية.

اللغة النهائية للمشروع: العربية (نص السرد عربي، وبرومبتات الصور بالإنجليزية).
النوع: True Crime عربي طويل. الأسلوب: وثائقي سينمائي مشوق محترم بطيء التصاعد.

==================================================
طريقة العمل (مرحلة واحدة متصلة — لا تنتظر اختياري)
==================================================

نفّذ الخطوات التالية في رد واحد متصل:

1) البحث والترشيح:
ابحث عن أقوى القضايا العربية الحديثة أو المفتوحة المناسبة لفيديو طويل، بشرط أن تكون:
- وقعت أو ظهرت إعلاميًا خلال آخر شهر إلى شهرين، أو
- قديمة نسبيًا لكن بها تطور موثق خلال آخر شهرين (إعادة فتح، حكم، جلسة، دليل، اعتراف، استئناف، قرار نيابة)، أو
- لا تزال مفتوحة/قيد التحقيق ولها مصادر حديثة موثوقة.
من العالم العربي فقط. ممنوع قضايا قديمة مغلقة بلا تطور حديث، وممنوع الاعتماد على شائعات.

اعرض جدولًا مختصرًا بـ 8–12 قضية مرشحة، لكل قضية سطر واحد يحوي: الرقم، الاسم، الدولة، آخر تطور (تاريخ تقريبي)، تقييم تريند/سرد/مصادر/حساسية (من 10)، وصلاحيتها لـ 40 دقيقة.

2) الترتيب والاختيار التلقائي:
رتّب أفضل 3 قضايا حسب: التريند، الحداثة/الاستمرار، قابلية السرد، CTR، Retention، توفر المصادر، انخفاض مخاطر التشهير.
ثم **اختَر تلقائيًا القضية الأقوى (رقم 1 في الترتيب)**، واكتب سطرًا واحدًا يوضح: «تم اختيار القضية رقم X لأن ...».

3) الإنتاج:
ابحث في القضية المختارة جيدًا من مصادر حديثة موثوقة، افهم التسلسل الزمني والشخصيات والتحقيقات والتحولات والنهاية، ثم أنشئ **ملف .json فعلي قابل للتحميل** بالاسم: `true_crime_arabic_case.json`.

==================================================
قاعدة الإخراج النهائية (صارمة)
==================================================
- لا تطبع محتوى JSON داخل الرد، ولا تستخدم بلوك كود لعرضه.
- المسموح فقط: جدول الترشيح + سطر الترتيب + سطر الاختيار + رابط تحميل الملف + جملة قصيرة تؤكد جاهزيته.
- لو البيئة لا تسمح بإنشاء ملف، أخبرني باختصار، ولا تطبع JSON إلا إذا طلبت ذلك صراحة.
- يجب أن يكون الملف صالحًا 100% (JSON سليم) وقابلًا للقراءة مباشرة بواسطة سكريبت الإنتاج.

==================================================
الضوابط (السلامة القانونية والأخلاقية)
==================================================
- لا تختلق حقائق. استخدم صياغات حذرة للتفاصيل غير المؤكدة: «بحسب تقارير متاحة»، «وفق ما ذُكر في التحقيقات»، «يُعتقد»، «قيل إن»، «لم يُؤكَّد رسميًا».
- لا اتهام مباشر لشخص لم يصدر ضده حكم. لا تشهير. احترام الضحايا والعائلات.
- لا وصف دموي أو صادم. لا صور لوجوه حقيقية. التزم بسياسات YouTube.
- لا تستخدم أسماء إلا إن كانت منشورة رسميًا في مصادر موثوقة، وبحذر.

==================================================
مواصفات المشاهد (scenes) — قلب الملف
==================================================
- scenes = Array من 28 إلى 40 مشهدًا، يغطّي مجملها 35–45 دقيقة.
- كل مشهد Object يحتوي بالضبط على المفاتيح: scene_number, narration, visual_prompt, motion_prompt, duration_seconds.
- narration: نص عربي كامل للتعليق الصوتي (تقريبًا 70–110 كلمة ≈ 30–45 ثانية)، طبيعي وسلس ووثائقي، يبني توترًا أو يكشف معلومة أو ينقل القصة للأمام. المشهد الأول Hook قوي جدًا في أول 20 ثانية. لا تكرار، لا لغة مبتذلة.
- visual_prompt: برومبت **إنجليزي** سينمائي رمزي جاهز لمولّد الصور، يصف لقطة رمزية تخدم نص المشهد (شارع عربي ليلي، باب شقة، ملف قضية، سيارة شرطة بعيدة، نافذة مضاءة، قاعة محكمة، خريطة مدينة، هاتف، مصباح شارع...). إلزاميًا: no real or identifiable faces, no gore, no graphic violence, symbolic only, dark cinematic, 16:9, consistent color palette across all scenes.
- motion_prompt: وصف **إنجليزي** قصير لحركة كاميرا/مشهد بسيطة (slow push-in, gentle pan...) بدون حشو جوّي مبالغ.
- duration_seconds: تقدير زمني للمشهد (عدد ثوانٍ).
- حافظ على ثبات بصري بين كل المشاهد (نفس الجو، نفس لوحة الألوان، نفس الأسلوب الرمزي).

==================================================
الصورة المصغرة (thumbnail_concepts) — 4 أفكار مختلفة
==================================================
كل فكرة Object فيه: concept (عربي)، visual_prompt (إنجليزي رمزي بلا وجوه/دم/سلاح صادم)، text_on_thumbnail (عربي قصير 2–5 كلمات)، ctr_reason (عربي).

==================================================
SEO (youtube_metadata)
==================================================
youtube_title (عربي قوي وغير طويل)، youtube_description (عربي طويل متوافق SEO)، hashtags (Array)، tags (Array، ضمن حدود يوتيوب، بلا كلمات تشهيرية)، pinned_comment (عربي)، chapters (Array من Strings مثل "00:00 - بداية اللغز").
